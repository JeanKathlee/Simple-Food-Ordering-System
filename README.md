# Simple-Food-Ordering-System
A web-based food ordering system that allows customers to browse a menu, add items to their cart, and place orders online. The system also includes an admin panel to manage menu items and update order statuses.
 
Features
-User Registration & Login
  Customers can create accounts and securely log in.
  
-Browse Menu
  View a list of available food items with descriptions and prices.
  
-Add to Cart
  Add selected food items to a shopping cart before checkout.
  
-Place Order
  Finalize and submit orders easily.
  
-Track Order Status
  Customers can view real-time order status:
    Pending
    Preparing
    Ready
    Delivered
    
-Admin Management
Admins can:
    Add, update, or remove menu items
    Manage incoming orders
    Update order statuses
